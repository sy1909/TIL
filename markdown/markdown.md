# 마크다운 배우기 / MD배우기

개발자의 문서양식 마크다운(markdown을 배웁니다)



## 제목

'#' 의 개수에 따라 제목의 중요도가 달라져요

h1 => 문서에서 가장 큰 제목

문서에 h1 과 h3만 있다 => 문서 잘못 작성



## 본문

본문은 그냥 적으면 본문이에요



## 리스트



### 순서있는 리스트

1. 식용유를 두른다
2. 파를 넣고 파기름을 낸다
3. 계란을 풀어서 익힌다
4. 간장을 볶는다

### 순서없는 리스트

- cli학습
- git복습
- md작성 익숙해지기
